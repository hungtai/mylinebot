#AI 影像專案: ILS CIM支援
   *模型建立平台 (ADJ-->UMS)
    1.現行ADJ 已符合建模需求, 可直接使用
    2.唯現行介面是以Defect影像作為設計, 有機化簡化部分如站點/機台等資訊後, 提供泛用影像建模平台 
   *Edge端推論系統(No Code)
    1.安防類No Code Edge: 包含電子圍籬(Human Detection,使用YOLO)及異物偵測(Abnormal Detection,使用PatchCore) 兩類.
    2.人員動素No Code Edge: 提供可自行設定動作順序, 停留時間等參數的No Code 程式.
   *跨廠間影像管理平台(ARGUS)
    1.建置跨廠區IP Camera 統一管理平台，整合即時影像、語意搜尋、AI 事件警報與訓練資料採集,及智慧分析平台.
    2.影像儲存在Edge端, 符合資安規範跨廠調用, 統一嫁接其他資訊系統, 減少購置NVR(錄影主機)硬體費用.
